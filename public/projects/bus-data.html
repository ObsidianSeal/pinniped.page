ROUTE MAPS

For ease of navigation, these maps simplify the routes into straight lines. This makes it very clear where certain routes diverge into lettered branches, have different termini at certain times, or exhibit other quirks, making it easy to identify your destination and see the stops that lead up to it. Each direction has its own map, so please pay attention to the file names.

These maps are made manually in Affinity Designer, so they will be added/updated very slowly.

Please report any inaccuracies!

DISCLAIMER
  • the information provided on this page should not be used as a substitute for official information provided by the London Transit Commission
  • the data processed by and shown on this page may not be 100% correct, as stated in the data’s Terms of Use

LIMITATIONS
  • some data is entered manually, so be sure to check the date that this page was last updated (it’s shown at the top of the page) or the GitHub commit history to see how up-to-date it is
  • this page assumes the Community Bus routes do not exist
  • rounding errors are almost certainly present

ABOUT THE MAP
  • you can change how the map looks and edit some of the map’s other settings by clicking the “sliders” icon in the top right or by pressing “SHIFT”
  • the data still automatically updates every 30 seconds, here the countdown is shown in the bottom right
  • hover over the map to see your cursor’s approximate latitude and longitude in the bottom left
  • the tiny dots are bus stops, all # of them
  • squares and circles are regular and articulated buses respectively
  • buses have a trail that shows where they have been recently
  • hover over a bus to learn more about it (the information from the main table is shown)
  • route colours are specified by London Transit as follows:
  • to see the routes’ hex codes, hover over the route numbers above or visit my route colours list and route map progress tracker
  • this page does not update if you leave, so there will be gaps in the bus trails if you leave and then return after more than 30 seconds
  • the bus map is technically an image, so you can save and/or copy it like you would any other
