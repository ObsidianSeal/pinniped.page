How to use fa-random

Controls/Keyboard Shortcuts:

  • ENTER or SPACE or the 'generate' button: get a new random icon
  • F: toggle fullscreen
  • ESCAPE: exit fullscreen
  • LEFT and RIGHT arrows: go back and forth between previously generated icons
  • C: copy a link to the current icon
  • BACKSPACE: reset icon (to url or first random)
  • H or SHIFT: toggle showing the menu
  • I: go to this help page

Notes:

  • The icon-?? text is useful for me, don't pay attention to it
  • The 'true' or 'false' after the dot shows if there's a regular icon to go with the shown solid icon
  • You can search for an icon by adding ?x=icon-name (no 'fa-') to the url
  • If the icon specified by the ?x=___ does not exist, default information and no icon will be shown
  • The above will also happen if you try to reset (with backspace) to an icon that does not exist
