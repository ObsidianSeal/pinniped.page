So. Many. Projects.

There have been so many amazing additions to the pinniped.page since the last changelog-type blog post, it’s actually unbelievable. I have added multiple games, fixed things that have been on my to-do list for some time now, made some additions to the main pages of the website, and more! So strap in, because this is going to be a long one.

December
  • More data is listed on the TicTacToe data page!
January
  • New project: wordlelele! Click the “???” in the top right of the wordlelele page for more information.
  • Various wordlelele updates.
  • If you don’t get the wordlelele, the answer is revealed and you can still set the next one.
  • Even more wordlelele updates: you can see when the word was last set, more FAQ answers.
  • Shrink to Fit can use decimal font sizes.
  • The wordlelele share message now includes the maximum number of guesses.
  • Fewer projects are now shown on the Projects page
  • The wordlelele now says “Loading...” before it has, you know, loaded.
  • The wordlelele now shows if there’s a new word!
  • Once you’ve submitted the next word on the wordlelele, the keyboard goes away.
  • The old VALORANT Points calculator has been removed, and the “legacy” button on the new version has been replaced with “back”.
  • More splash texts on the homepage!
February
  • The wordlelele’s “???” page now has a maximum width.
  • You can now click my name (Obsidian_Seal) at the bottom of the wordlelele’s “???” page to go to the About page
  • wordlelele themes! But only from the URL for now.
  • A couple splash texts on the homepage have been removed.
  • Quick navigation improvements.
  • CSS changes.
March
  • New project: Graph Theory EXPLAINED! I made this mostly for school.
  • New project: Finite Craft! This project has been clicked hundreds of times on Google!
  • To explain how to play Finite Craft, I made a how-to page.
  • There’s no notice on the homepage...for now.
  • The titles of blog posts (like this one) are now also present in the page title (what you see on the tab or in your history).
  • Sometimes it was impossible to leave the wordlelele (from the “exit” button), but I think I’ve fixed it.
  • The description of Words has been edited.
  • Better link wrapping, specifically for the sources listed in Graph Theory EXPLAINED.
  • All single and double quotes have been made “fancy” with HTML entities.
  • Improved external embed (OGP) consistency.
  • Finite Craft how-to improvements.
  • New project: crosswords!
  • crosswords updates, loading screen, open to latest puzzle by default.
April
  • The error 404 page was broken (in some cases) but I fixed it.
  • Improved redirects for short URLs!
  • New project: Think Four! This completes my trilogy of New York Times games.
  • CHESS/BOARD (FEN preview): keyboard/mouse input fixes, and some other things. This had been on my to-do list since September 1st.
  • At least a couple projects have had icons changed.
  • The Projects/Projects+, Blog/Blog+, and About pages now have nice looking titles!
  • The navigation bar now has the Obsidian_Seal head in place of “Home”, the first change to navigation bar since I made it over two years ago.

Wow. That’s a lot. Five new projects, countless updates, major changes to EVERY non-project page. And there’s lots more to come. For the time being, I would love to hear any feedback you have.
