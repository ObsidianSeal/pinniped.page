Even More (Super Cool) Changes

This past week has brought even more changes to the pinniped.page:

  • Projects+ accepts Regular Expression searches.
  • Projects and Projects+ now actually have a search bar! With tag selection!
  • The search bar fits on small devices too.
  • Search feature: fixed problems with the “no results” message on small devices, made it so the placeholder is shortened on small devices, and more.
  • The tags that appear on the Projects+ page now also appear on the Projects page!
  • The GitHub repository (ObsidianSeal/pinniped.page) has gone public.
  • Changes to some of the Bus Ride Bingo squares, complete rewrite coming soon.
  • More subtitles on the homepage.
  • Remember those tags on the Projects and Projects+ pages? Now they have tooltips! The tooltips specify what was changed and when.
  • Type (a.k.a. keyPressTest) has a different font, clearer description, more detailed key information, and more?
  • The rate-puzzle stars on Think Four change colour on hover to match what would happen if you clicked. The CSS behind that is both amazing and awful.

Thanks for reading. While you’re here, and since it has been on my to-do list for so long (June 11th, 2023), here are two interesting animation tools that I have come across.

  1. Google Earth Studio — it’s Google Earth with the added ability to make complex keyframe animations. As of writing this, you have to answer a couple questions to request access.
  2. Motion Canvas — really nice 2D animation created entirely through code.
